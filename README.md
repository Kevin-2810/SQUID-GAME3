# 🎮 Squid Game Challenge Website

An interactive and immersive **Squid Game-inspired web application** built using **HTML, CSS, and JavaScript**. This project recreates the thrilling atmosphere of the popular Squid Game series with multiple game modes, animated backgrounds, cinematic UI, and detailed game descriptions.

## 🚀 Features

- 🎯 Interactive Main Menu System
- 🎬 Dynamic Video Backgrounds
- 🌟 Modern Gaming UI Design
- 🎮 Multiple Squid Game Challenges
- 🔥 Smooth Animations & Transitions
- 📱 Responsive Design
- ⌨️ Keyboard Navigation Support
- 🎨 Neon-Themed Visual Effects

## 🎲 Available Game Modes

- Red Light, Green Light
- Six Legs Challenge
- Mingle
- Hide N' Seek
- Jump Rope
- Dalgona Candy
- Glass Bridge
- Marbles

Each game mode includes:
- Unique background video
- Detailed game description
- Interactive selection effects
- Cinematic experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts (Orbitron)
- Video Background Integration
